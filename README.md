# Curso de Node JS (InfomaticaDP)

## Iniciar un proyecto de Node

```bash
$ npm init
```

### Instalar y desintalar modulos en el proyecto.

##### Instalar

```bash
$ npm i <nombre_modulo>

$ npm i bootstrap
```

##### Desinstalar

```bash
$ npm uninstall <nombre_modulo>

$ npm uninstall bootstrap
```

### Ejecutar proyecto

```bash
$ node  <archivo.js>
```
